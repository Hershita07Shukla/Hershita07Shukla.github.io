Multi-Sensor Fusion for Vehicle Detection and Speed Tracking using Camera and RADAR (in collaboration with Suzuki Motor Corporation)

Python Radar Camera Fusion RetinaNet Deep Learning

I worked on developing a modified camera and radar fusion algorithm for vehicle detection and speed tracking as a part of my MTech thesis in IITH-TiHAN. The project aimed to enhance single-stage detection accuracy using advanced fusion techniques.

I worked on developing a modified camera and radar fusion algorithm for vehicle detection and speed tracking at Suzuki Motor Corporation. The architecture was based on a RetinaNet backbone, where we experimented by increasing network width, depth, and introducing skip connections. Based on comparative study of multiple deep learning models, we proposed a fusion framework that improved object detection, feature extraction, and sensor fusion. This system aimed to ensure high reliability in distance estimation and real-time vehicle behavior monitoring using combined camera and RADAR inputs.
